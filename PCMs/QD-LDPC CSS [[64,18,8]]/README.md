This folder contains the PCMs for the QD-LDPC CSS code [[64,18,8]].
The code is not degenerate, in the sense that the quantum d_min is equal to the two classical d_min.
The stabilizer generator weight is 8.
QD[[64,18,8]]_P_Z and QD[[64,18,8]]_P_X are the two protogrpahs 
referring QD[[64,18,8]]_H_Z and QD[[64,18,8]]_H_X, repsectively.
